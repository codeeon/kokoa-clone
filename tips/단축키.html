• 맥 기준에서, 브라우저 새로고침 키 = command + r // 윈도우는 ctrl+r 아마도
  • tab을 줄이는 건 shift + tab
  • 태그 [열린 부분], [/닫힌 부분] 동시 선택 = 한 태그 위에 커서 두고 command + d -> 동시 선택 가능 (윈도우는 ctrl+d), 커맨드 디에 방향키 사용 가능
  • 해당 영역 주석 처리 command + / (윈도우 ctrl+/)
  • 맥에서 백팁(백쿼터,`)을 사용하는 법 = option(⌥) + ₩ (또는 옵션 + \_백슬래시 - 기기마다 다른 듯)
  • . + _ + tab -> div class 설정
