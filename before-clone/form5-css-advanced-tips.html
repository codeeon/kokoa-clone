Transitions

트랜지션 애니메이션을 만든다. 호버 등에 쓰면 안 되고, element(요소)에다 써야 한다.
주로 ease-in-out을 이용한다. cubiz-bezier를 이용하면 나만의 커브(속도 조절)를 설정하여 사용할 수 있다.

Transformations

트랜스폼을 이용한다. hover에 쓰면, 해당 효과가 나타나며, 트랜지션과도 함께 쓸 수 있다.(트랜지션은 본 태그)
rotate(숫자deg)를 이용하여 XYZ축을 deg(각도) 단위로 변경 가능, skew(숫자deg)를 이용하면 기울기가 가능하다.
scale(1_디폴트)값을 이용하여 크기 조정도 가능하다.
이외에도 matrix 등 여러 가지 있음. 항상 트랜지션과 함께 쓰는 것을 추천. 검색할 땐 css transform mdn


Animations

말 그대로 애니메이션이다. 키프레임부터. 마우스 호버링 없이 사이트 접속하자마자 발생하는 애니메이션이다.
@keyframes를 이용해 애니메이션 변수를 만들고 트랜스폼 값을 넣는다. from {} to {}로 넣거나 0%{} 50%{} 100%{} 등으로 나눠서 트랜스폼을 넣는다.
적용하고 싶은 곳에 animation: 애니메이션 변수를 기입한다(트랜지션처럼). 트랜지션 값 맨뒤에 infinite를 넣으면 무한반복한다.
검색어는 [CSS animations], Animista
애니메이션이나 트랜스폼이나 애프터이펙트와 효과가 동일한 듯, opacity scale 등
